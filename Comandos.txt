========================================================== COMANDOS ==========================================================

° npx create-react-app [nombre_proyecto] --template typescript: Comando utilizado para crear un proyecto de React.

° npm install --save-dev typescript @types/node @types/react @types/react-dom @types/jest @types/react-router-dom: Comando utilizado para instalar la ultima versión del paquete para enrutar los componentes.

° npm install bootstrap --save: Comando utilizado para instalar la ultima versión de Bootstrap.

° npm run start: Comando utilizado para hacer correr el servidor de React. El servidor corre por defecto en el navegador con la dirección localhost:3000
